# Placement Prediction and Career Guidance System

## Project Overview

The Placement Prediction and Career Guidance System is a web-based application developed using Python Flask and MySQL. The system helps predict whether a student is likely to be placed based on factors such as CGPA, aptitude score, communication skills, internships, and technical skills.

In addition to placement prediction, the application provides career guidance recommendations that help students improve their employability and prepare for placement opportunities.

## Features

- Student information collection through a web form
- Placement prediction (Placed / Not Placed)
- Career guidance recommendations
- Internship and skills tracking
- MySQL database integration
- User-friendly interface

## Technologies Used

- Python
- Flask
- MySQL
- HTML
- CSS

## Workflow

1. Student enters personal and academic details.
2. Data is sent to the Flask backend.
3. Placement prediction logic is applied.
4. Career guidance suggestions are generated.
5. Student details and prediction results are stored in MySQL.
6. Results are displayed on the webpage.
